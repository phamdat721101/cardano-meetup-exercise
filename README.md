# cardano-meetup-exercise

## Overview
Welcome to the cardano-meetup-exercise repository. This repo contains resources and instructions for hands-on lab sessions focusing on various aspects of the Cardano blockchain.

## Lab Overview
In this lab, participants will learn how to:
1. Set up a Cardano node.
2. Interact with the Cardano blockchain through tools and frameworks.
3. Understand the basics of smart contract development on Cardano.
4. Participate in transactions and learn about decentralized finance (DeFi) opportunities.

## Instructions
1. **Clone the Repository**: Begin by cloning this repository to your local machine:
   ```bash
   git clone https://github.com/phamdat721101/cardano-meetup-exercise.git
   cd cardano-meetup-exercise
   ```

2. **Set Up the Environment**: Ensure that your development environment is configured according to the lab specifications (Haskell, Cardano tools, etc.).

3. **Follow the Labs**: Navigate through the folders:
   - [meetup-1/SCREENSHOTS_GUIDE.md](meetup-1/SCREENSHOTS_GUIDE.md)
   - [meetup-2/MEETUP_2_GUIDE.md](meetup-2/MEETUP_2_GUIDE.md)

4. **Get Help**: If you encounter issues, reach out in the community chat or consult the FAQ.

## Conclusion
This lab aims to provide hands-on Cardano experience. Happy learning!
